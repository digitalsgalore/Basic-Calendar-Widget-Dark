# Calendar Widget (Dark Mode)

Open `index.html` in a browser to view the widget. The page background is `#191919` and the widget background is `#1E1E1D` with a thin white border.

To run a quick local server (recommended) from the project folder:

Windows / PowerShell:

```bash
python -m http.server 8000
```

Then open http://localhost:8000 in your browser.
